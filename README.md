============

* 盾牌造型表示我们是立志于保护防卫型的组织。

* 屯卦画符亦表示我们来自中国的文化，我们的文化是中华文化。

* 锤子画符表示我们我们是社会主义者，坚持社会主义是我们的底线。

* 三角形和js字符表示我们使用JavaScript以构建。
